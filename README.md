## NodeJS REST API - SwaggerUI template

This project will run an ExpressJS Rest API App with SwaggerUI API Documentation. 

#### Requirement

1. Database running on localhost, (default is Mysql)

#### Installation

1. Clone this repository
```
git clone git@github.com:aridwan/nodejs-express-swagger.git
```

2. Install dependency
```
npm install
```

3. Create Database
```
npm run db:create
```

4. Run migration
```
npm run db:migrate
```

5. Run App
```
npm start
```
